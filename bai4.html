Bài tập 4 - Bài 4: Giao tiếp và hợp tác trong môi trường số

🎯 1. Mục tiêu của bài tập

Làm quen và ứng dụng các hệ thống quản lý mã nguồn Git, nền tảng lưu trữ GitHub, cũng như các công cụ quản lý dự án trực quan (Trello/Kanban) và truyền thông nhóm (Slack/Discord) để phát triển và hoàn thiện dự án nhóm trong môi trường số.

🛠️ 2. Quá trình thực hiện

Quy trình làm việc nhóm trực tuyến được cụ thể hóa bằng các bước sau:

  • ✔ Bước 1: Thiết lập Git & GitHub: Khởi tạo kho lưu trữ local (Local Repository), tạo kết nối và đẩy mã nguồn lên GitHub. Thực hành các nhánh phát triển (branching) để phân chia nhiệm vụ code độc lập cho các thành viên.
  • ✔ Bước 2: Hợp tác và Giải quyết Xung đột: Tạo các Pull Request (PR) để gộp mã nguồn vào nhánh chính. Thực hành rà soát code (code review) nhóm và trực tiếp xử lý các lỗi xung đột code (conflict resolution) phát sinh.
  • ✔ Bước 3: Quản lý công việc bằng Trello: Xây dựng bảng công việc Kanban trên Trello, phân bổ các thẻ nhiệm vụ dưới dạng To-Do, In Progress và Done để theo dõi trực quan tiến độ của dự án.
